# Mini Service Finder

A clean, map-based mini project for discovering nearby public services, with a new work-from-home friendly category.

## Features
- Interactive Leaflet map
- Search and category filters
- Suggest new services with local storage persistence
- Home-based work support options

## Run locally
1. Open Index.html in a browser, or serve the folder with a simple static server.
2. For example: `python -m http.server 8000`
3. Visit http://localhost:8000/
